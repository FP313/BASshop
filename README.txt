Name project "BAS"

The purpose of the BAS project is to develop a web application that will allow people to conveniently sell
and buy accounts of any kind, from gaming accounts of popular companies (Steam, Riot, EA, EpicGames, and others)
to social media and niche gaming accounts (WhatsApp, Telegram, YouTube, TikTok).


Necessary dependencies sqlite3 and jwt(jsonwebtoken)

launch commands ..\backend node app.js
